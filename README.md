# Manus 复刻（前端原型）- v0.2 Beta

这是一个轻量级的 **Manus 风格工作台** 原型，面向大众用户展示 AI 任务编排体验，聚焦以下能力：

- 左侧任务列表（待办 / 进行中 / 已完成）+ 状态筛选 Tab
- 全球大模型厂商能力图（推理 / 工具调用 / 速度）
- Skills 能力标签（可编排能力模块）
- 通讯平台连接（Slack / Discord / Teams / Telegram / WhatsApp / 飞书 / 钉钉）
- Google 账号接入（前端模拟 OAuth 接入状态）
- 中间 AI 对话区 + 右侧执行日志（模拟自动化执行轨迹）

> 说明：这是 UI/交互原型，不连接真实模型与外部工具。

## 发布信息

- 当前发布：`v0.2 Beta`（测试版）
- 发布日期：2026-04-09
- 发布范围：前端静态原型（无后端依赖）

## 本地运行

```bash
python3 -m http.server 8080
```

然后访问：

- <http://localhost:8080>

## 目录结构

- `index.html`：页面骨架（三栏布局 + 任务筛选）
- `styles.css`：样式系统（深色主题 + 能力条可视化）
- `app.js`：交互逻辑（任务流转、聊天、日志、平台连接、Google接入）
